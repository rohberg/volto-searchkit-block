The dictionary_decompounder token filter is used to split compound words into their constituent parts based on a predefined dictionary.

See https://opensearch.org/docs/latest/analyzers/token-filters/dictionary-decompounder/ 


The keyword_marker token filter is used to prevent certain tokens from being altered by stemmers or other filters.

See https://opensearch.org/docs/latest/analyzers/token-filters/keyword-marker/
